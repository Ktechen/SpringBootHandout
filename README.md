| annotations   					| Beschreibung  | 
| ------------- 					|:-------------:|
| @RestController    					|Der Controller verarbeitet RESTful services	| 
| @GetMapping/@RequestMapping(method=GET) 		|HTTP-Anfragen z.B GET/POST/PUT/DELETE		|  
| @Bean 						|     						|   
| @Component						|     						|   
| @Configuration					|     						|   
| @Autowired						|     						|
| @RequestParam(value = "name", defaultValue = "World")	|     						|
